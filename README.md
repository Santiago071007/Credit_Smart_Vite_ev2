#  CreditSmart — Simulador y Solicitud de Créditos

-**Nombre** = David Santiago Cantor Torres

- Con este uso nos permite realizar validaciones de formularios, y asi como filtrar por tasa y categoría, y realizar solicitudes completas mediante un formulario controlado, todo esto con la implementacion de tecnologias como:

- # Tecnlogias implementadas:

- Estados y validaciones con **useState**
- **React Router DOM**
- **CSS personalizado**
- **Vite**

- Funciones principales de el sistema:

- Home.jsx Page ( es el index.html convertido a un jsx): mostrar los creditos disponibles de credit smart, como sus header y los mensajes que contiene la pagina como "Tu crédito ideal en segundos
Simula, compara y elige el mejor crédito para impulsar tus sueños" y asi mismo las tarjetas o cards se renderizan dinámicamente desde un array usando props

- Simulador.jsx page (es el simulador.html convertido a jsx): Permite buscar créditos por nombre, asi como filtros que son dinamicos permitiendo buscarlos por tasa de interes, los montos y demas. Asi como un boton de limpiar los filtros que se aplicaron anteriormente y por ultimo Tarjetas actualizadas automáticamente según la búsqueda.

- Pedir.jsx page ( solicitar.html convertido a jsx): Estados controlados para cada input, permite validaciones de campos como el email, cedula, y el monto minimo de el credito, despues el boton de limpiar el formulario, donde debe limpiar todo lo diligenciado por el usuario. por ultimo enseña un resumen de la solicitud para revisar si necesita implementar cambios en la solicitud

- Componentes principales:

- **App.jsx** : controla las rutas de credit smart
- **Home.jsx** : muestra la lista de creditos disponibles
- **Simulador.jsx** : Buscador y filtros dinámicos
-**Pedir.jsx (Solicitar)** : Formulario controlado
-**CreditCard.jsx** : Tarjeta reutilizable para mostrar créditos
